Citações e Códigos

Código-Fonte / Pré-Formatação

O comando document.getElementById('elemento') é escrito em linguagem JavaScript. O formato de código é escrito com a tag <code>.

Exemplo de código em linguagem Python: 


num = int(input("Digite um número: "))
if num % 2 == 0:
print(f"O número {num} é PAR!")
else:
print(f"O número {num} é ÍMPAR!")
print("Fim do programa!")
